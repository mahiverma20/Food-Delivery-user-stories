# Food-Delivery-user-stories

**Sandspace Technologies Private Limited**

This repository contains 20 user stories for a Food Delivery App. All stories are written from the end user's perspective following the standard Agile format.

**Format:** As a [user], I want to [action], so that [benefit].

**Modules Covered:** Onboarding, Restaurant Discovery, Menu & Ordering, Checkout & Payment, Order Tracking, Post-Order, Account & Profile.

**File:** `UserStories_FoodDeliveryApp.docx`
